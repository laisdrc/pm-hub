# PM Hub

Uma plataforma web de gerenciamento de projetos desenvolvida para centralizar informações, monitorar entregas e otimizar a tomada de decisões em um único ambiente.

Inspirado nas melhores práticas de gestão, o PM Hub consolida tudo o que um gerente de projetos precisa: tarefas, cronogramas, riscos, equipe e indicadores de desempenho. Uma solução projetada para ser intuitiva, altamente visual e totalmente orientada a dados.

---

## Funcionalidades previstas

| Funcionalidade | Descrição |
| :--- | :--- |
| **Autenticação** | Cadastro e login de usuários |
| **Kanban** | Quadro de tarefas por projeto |
| **Cronograma** | Linha do tempo de projetos |
| **Gestão de riscos** | Registro e acompanhamento de riscos |
| **Gestão de equipe** | Membros, papéis e responsabilidades |
| **Dashboard** | Indicadores e métricas via Power BI |

---

## Tecnologias

### Frontend
- **React** — Interface de usuário
- **React Router** — Navegação entre telas

### Backend e Banco de Dados
- **Supabase** — Autenticação, banco de dados

### Dados e BI
- **Power BI** — Visualização de indicadores e dashboards

### Qualidade
- **Cypress** — Testes automatizados end-to-end
- **SonarCloud** — Análise estática de qualidade de código

### Infraestrutura
- **Azure** — Hospedagem e publicação da aplicação

## Instalação - Aprenda a rodar o projeto localmente

# 1. Clonar o repositório 
git clone https://github.com/laisdrc/pm-hub.git

# 2. Entrar na pasta do projeto 
cd pm-hub

# 3. Instalar as dependências 
npm install

# 4. Iniciar o projeto em modo de desenvolvimento
npm run dev
