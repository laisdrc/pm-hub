# 🚀 'PM Hub'
Sistema web de gerenciamento de projetos.

## 📌 Objetivo
Desenvolver uma plataforma capaz de centralizar informações de projetos, acompanhar atividades, gerenciar riscos e visualizar indicadores em um único ambiente.

## 🛠️ Tecnologias
- React
- Supabase
- Azure
- Power BI
- Cypress
- SonarCloud
