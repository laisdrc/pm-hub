# Termo de Abertura do Projeto (TAP)
// O que é o projeto?

## Nome do Projeto
PM Hub

## Gerente do Projeto
Laís Câmara

## Data de Início
04 de Junho de 2026

## Objetivo
Desenvolver uma plataforma web de gerenciamento de projetos que permita centralizar informações, acompanhar atividades, gerenciar riscos e visualizar indicadores de desempenho em um único ambiente.

Além do desenvolvimento técnico, o projeto tem como objetivo aplicar conceitos de Gerenciamento de Projetos estudados durante a preparação para a certificação CAPM.

O PM Hub será utilizado como projeto de portfólio para demonstrar conhecimentos em desenvolvimento de software, gestão de projetos, testes automatizados e análise de dados.

## Escopo
O sistema contará com os seguintes módulos:

* Kanban de tarefas
* Cronograma do projeto
* Gestão de riscos
* Dashboard de indicadores
* Cadastro e gestão da equipe

## Tecnologias
* React
* Supabase
* Azure
* Power BI
* Cypress
* SonarCloud

## Principais Entregas
1. Estrutura inicial do sistema
2. Módulo Kanban
3. Módulo Cronograma
4. Módulo de Gestão de Riscos
5. Dashboard de KPIs
6. Testes automatizados
7. Publicação da aplicação

## Restrições
* Desenvolvimento realizado por uma única pessoa.
* Tempo disponível limitado devido às atividades acadêmicas e profissionais.
* Utilização prioritária de ferramentas gratuitas ou disponibilizadas por licença estudantil.

## Critérios de Sucesso
* Sistema publicado e acessível online.
* Principais módulos implementados e funcionais.
* Aplicação de práticas de Gerenciamento de Projetos durante o desenvolvimento.
* Disponibilização do código-fonte e documentação no GitHub.
